# 下载

## X86_64

```bash
# Git 克隆
git clone https://github.com/langong-dev/Zff.git
```

## ARM

```bash
# Git 克隆
git clone https://github.com/langong-dev/Zff.git

# 编译
g++ zff_main_en.cpp -o zff-en && g++ zff_main_ch.cpp -o zff-ch
```

# 运行

```bash
# 英文
cd ~/Zff && ./zff-en

# 中文
cd ~/Zff && ./zff-ch
```
