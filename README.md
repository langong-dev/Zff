# Install

## In X86_64

```bash
# Git Clone
git clone https://github.com/langong-dev/Zff.git
```

## In ARM

```bash
# Git Clone
git clone https://github.com/langong-dev/Zff.git

# Compile
g++ zff_main_en.cpp -o zff-en && git zff_main_ch.cpp -o zff-ch
```

# Run

```bash
# English
cd ~/Zff && ./zff-en

# Chinese
cd ~/Zff && ./zff-ch
```
