# Install

## AUR user (ArchLinux)

```bash
yaourt -S zff  ## you can also use other app, like yay, pacaur...
```

## Others

```bash
git clone https://github.com/ohzff/Zff
cd Zff
sudo bash install.sh
```

# Add user

```bash
zff init
```

# Run

```bash
zff
```

# Update

```bash
cd /usr/share/ohzff-zff
sudo git pull
sudo g++ zff_main_en.cpp -o /usr/bin/zff
```

Aur users

```bash
yaourt -Syu
```
